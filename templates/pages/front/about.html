A propos

GitHub Copilot

le site est à des fins de recherche sur l'utilisation de GitHub Copilot sur les sites développés avec Django pour évaluer la pertinence des résultats.

Github Copilot a été utilisé dans différents environnements pour la réalisation de tests de pertinence.

  • Fichiers html.
  • Fichiers python.
  • Template Django.

L'outil utilisant une l'intelligence artificielle développé par Github Et OpenAI pour aider les développeurs, en completant automatiquement le code.

Conclusion

En conclusion Github Copilote est un outil qui permet d'aider les développeurs dans le temps en proposant des actions logiques dans le contexte de développement, il peut permettre d'aider à l'ajout de nouvelles fonctionnalités pas forcément pensé lors de la conception d'un projet.
